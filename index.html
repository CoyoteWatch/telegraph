Как создать статью с логгером

  1. 1
    Для создания статьи с логгером, требуется зайти в бота @CoyoteWatch_bot
  2. 2
    Введите команду /blog – бот начнёт процесс создания статьи
  3. 3
    Укажите заголовок (например: «Новый бот для создания логгер-ботов»)
  4. 4
    Введите имя автора (например: CoyoteWatch)
  5. 5
    Напишите текст статьи (например: «Привет, мир!»)
  6. 6
    Добавьте кодовое слово – это поможет идентифицировать лог из конкретной статьи, особенно если у вас их много

Выгрузка логов

Также имеется функция выгрузки логов. Нажимая на кнопку "Выгрузить все переходы", вам отправится HTML файл с удобным интерфейсом для просмотра логов вашей статьи.
